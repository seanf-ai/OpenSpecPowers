# Support

## Get Help

- Usage and setup questions: open a GitHub issue with context and logs.
- Bug reports: use the Bug Report issue template.
- Feature requests: use the Feature Request issue template.

## Before Opening an Issue

- Confirm you're on the latest release.
- Run `openspecpowers --version`.
- Include your OS, Node.js version, and reproduction steps.

## Commercial / Team Support

If you need dedicated support or onboarding for teams, use the contact details in [README.md](README.md).
